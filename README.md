#### 功能和特色

* 适合写作的设计；
* 炫酷的侧滑导航栏设计；
* 文章页打赏功能；
* 紧凑的按钮设计；
* 响应式设计；
* Pjax技术；
* 代码高亮；
* 又大又长的首页头图；
* 不丰富的社交按钮，只有4个；
* 文章页图片放大功能；
* 支持嵌入第三方评论；
* 支持Ghost、Typecho、Hexo这三个平台；
* Alpha版本，更多功能和优化请期待，谢谢！
